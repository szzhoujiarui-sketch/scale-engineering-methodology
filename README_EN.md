# SCALE Engineering Methodology v1.0.8

> Three-tier annotation — 🟢 Core 🟡 Advanced 🔵 Deep. A methodology for cultivating Agent engineering discipline.

## Core Rules

| Rule | One-liner |
|------|-----------|
| Anti-Hallucination | Don't describe file contents you haven't read; don't claim tests passed that you haven't run |
| Anti-Laziness | Read before writing, verify after changing |
| Honest Delivery | ✅ Done / 🔄 Verified / ❌ Unverified |
